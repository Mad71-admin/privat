# Wissen

Ein Sammelbecken für Vorlieben, laufende Projekte, langfristige Ziele und persönliche Wünsche.

## Inhalte
- **Vorlieben**: Was ich mag (Essen, Musik, Hobbys)
- **Projekte**: Private Vorhaben und deren Status
- **Ziele**: Was ich erreichen möchte
- **Wünsche**: Materielle oder immaterielle Wünsche
