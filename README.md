# Markus' Private Wissensdatenbank

Diese Wissensdatenbank dient als zentrales Gedächtnis für Markus' private Belange. Sie ist darauf ausgelegt, die Zusammenarbeit mit KI-Assistenten (wie Manus) zu verbessern, indem sie Kontext zu Markus' Person, seinem Umfeld, seinen Vorlieben und Projekten bietet.

## Struktur

- **00_UEBER_MARKUS/**: Wer ich bin, was ich kann (Persönliches Profil, Fähigkeiten)
- **01_MEINE_PARTNER_FAMILIE_UND_FREUNDE**: Informationen über wichtige Personen im Leben von Markus
- **02_MEINE_HOOD**: Informationen über das Wohnumfeld und lokale Gegebenheiten
- **03_MEIN_SOCIAL_MEDIA**: Präsenz und Aktivitäten in sozialen Medien
- **04_JOYCLUB**: Spezifische Informationen und Präferenzen bezüglich Joyclub
- **05_URLAUB**: Reiseerfahrungen, geplante Urlaube und Vorlieben
- **06_WISSEN/**: Vorlieben, Projekte, Ziele und Wünsche

## Nutzungshinweis

Diese Datenbank wird kontinuierlich erweitert. Neue Erkenntnisse aus abgeschlossenen Tasks werden hier dokumentiert, um eine langfristige und personalisierte Unterstützung zu ermöglichen.
